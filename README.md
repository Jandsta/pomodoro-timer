# pomodoro-timer
A beginner Python Pomodoro timer with countdown in MM:SS format.

## What It Is
A beginner Python project created in Vim on Linux Mint.
It's a simple Pomodoro timer where you can enter your work and break times.
The program validates your input and counts down in real time, displaying minutes and seconds.

## How to Use
1. Clone or download this repository.
2. Make sure Python 3 is installed.
3. Open terminal and navigate to the project folder.
4. Run the program:
```bash
python3 pomodoro.py
```
5. Enter your desired work and break minutes.
6. Watch the countdown for your pomodoro session.

Features:
-Input validation (ensures positive numbers)
-Converts minutes to seconds automatically
-Real time countdown using time.sleep()
-Displays countdown in MM:SS format

## Why I Made This
To practice Python basics like:
-Variables and input
-Loops
-Input validation
-Using the time module for real-time functionality
